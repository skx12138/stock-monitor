# A股行情监控 + 微信信号提醒

基于 Python + AKShare 的 A股实时行情监控工具，检测技术指标信号并通过企业微信推送通知到手机。

## 功能

- 📊 实时获取 A股行情数据
- 🔴 均线金叉/死叉信号检测（MA5 / MA20）
- 📱 信号触发时通过企业微信推送到手机
- ⏱ 盘中自动轮询（可配置间隔）

## 快速开始

### 1. 安装依赖

```bash
cd stock-monitor
pip install -r requirements.txt
```

### 2. 配置企业微信机器人（可选但推荐）

1. 打开企业微信 → 创建一个群聊 → 群设置 → 群机器人 → 添加
2. 复制 Webhook 地址
3. 粘贴到 `config.yaml` 的 `wechat_webhook` 字段

### 3. 配置自选股

编辑 `config.yaml`，在 `stocks` 下添加你想监控的股票：

```yaml
stocks:
  "600519": "贵州茅台"
  "000858": "五粮液"
  "601318": "中国平安"
```

### 4. 运行

```bash
python run.py
```

## 配置说明

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `wechat_webhook` | `""` | 企业微信机器人 Webhook 地址 |
| `interval_seconds` | `60` | 轮询间隔（秒） |
| `ma_short` | `5` | 短期均线周期 |
| `ma_long` | `20` | 长期均线周期 |
| `lookback_days` | `30` | 拉取历史K线天数 |

## 交易时间

系统自动识别 A股交易时间：
- 上午 9:30 - 11:30
- 下午 13:00 - 15:00
- 周末自动跳过

## 扩展

想添加更多信号？在 `src/signals.py` 中新增检测函数，然后在 `check_signals()` 中调用即可。
