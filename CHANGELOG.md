# 更新日志

## 2026-06-26 (周五) — 全面优化

### 交易策略优化
- **移动止盈**: 盈利6%激活 → **4%激活**，更早锁定利润
- **分批止盈**: 12%/20%/30% → **8%/15%/25%**，降低止盈门槛
- **ATR止损**: 2.5倍 → **2.0倍**，更严格止损
- **固定止损**: 保持 **-8%**
- **追涨限制**: 日内涨>4%不买，>2%减半
- **深跌机会**: 跌>5%允许半仓买入

### 加仓策略精简
- **金字塔加仓** (15%/10%/5%) ✅ 保留
- **回踩MA10/MA20加仓** (10%) ✅ 保留
- **放量突破加仓** (10%) ✅ 保留
- **RSI超卖加仓**: 15% → **8%** ✅ 仓位减半
- **大盘共振加仓** ❌ 移除
- **亏损摊平** (≤5%) ✅ 保留

### 跌停处理
- 跌停+预测看涨 → **1.5倍仓位抄底**
- 跌停+预测看跌 → **持仓等待反弹，不割肉**

### 尾盘买入优化
- **追涨模式**: 涨幅1%~5% + 多头排列 + RSI 40~65
- **低吸模式**: 跌幅2%~8% + MA20支撑 + RSI 20~60
- 尾盘加仓: 已持仓评分≥50加仓10%
- 无候选时列出全市场扫描

### 推送优化
- 启动通知: **9:30仅一次**，显示所有监控股票实时行情
- 信号播报: **每30分钟一次**（之前每次触发都推）
- 异动提醒: 波动>**2.5%**触发（之前>1.5%）
- 大盘风险预警: 涨>1.5%/跌>1%每小时一次
- 明日预测: 14:50~15:00每分钟推送，含整体趋势
- 每日交易总结: 15:00推送买卖明细+持仓
